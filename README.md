# Лабиринт Сокровищ

Текстовая приключенческая игра на Python, где вы исследуете таинственный лабиринт в поисках сокровищ!

## Описание проекта

**Лабиринт Сокровищ** — это интерактивная текстовая игра в жанре text adventure. Вы играете за искателя приключений, который попадает в древний лабиринт, полный загадок, ловушек и сокровищ. Ваша цель — найти ключи, решить головоломки и добраться до заветного сундука с сокровищами!

### Особенности игры

- **6 уникальных локаций**: от тёмного входа до комнаты с сокровищами
- **Система загадок**: решайте головоломки для получения наград
- **Случайные события**: встречи с существами, находки и ловушки
- **Система инвентаря**: собирайте и используйте предметы
- **Закрытые комнаты**: найдите ключ для доступа к сокровищам
- **Ловушки**: будьте осторожны в опасных местах!

## Как играть

### Доступные команды

| Команда | Короткая форма | Описание |
|---------|----------------|----------|
| `help` | `h` | Показать список команд |
| `look` | `l` | Осмотреться в текущей комнате |
| `go <direction>` | `g <direction>` | Перейти в указанном направлении |
| `north`, `south`, `east`, `west` | - | Перемещение без команды `go` |
| `take <item>` | `t <item>` | Взять предмет из комнаты |
| `inventory` | `i` | Показать содержимое инвентаря |
| `use <item>` | `u <item>` | Использовать предмет из инвентаря |
| `solve` | `s` | Решить загадку или открыть сундук |
| `quit` | `q` | Выйти из игры |

### Направления

- `north` (север)
- `south` (юг)
- `east` (восток)
- `west` (запад)

### Пример игрового процесса

```
> look
== ENTRANCE ==
Вы в темном входе лабиринта...
Заметные предметы: torch
Выходы: north, east

> take torch

> north
== HALL ==
Большой зал с эхом. По центру стоит пьедестал с запечатанным сундуком.
Выходы: south, west, north
Кажется, здесь есть загадка (используйте команду solve).

> solve

Ваш ответ: 10

За решение загадки вы получили золотую монету!
```

## Цель игры

Ваша задача — добраться до комнаты с сокровищами (`treasure_room`) и открыть сундук. Для этого вам понадобится:

1. Найти ключ от сокровищницы (в `trap_room`)
2. Решать загадки для получения наград
3. Избегать или преодолевать ловушки
4. Открыть сундук с сокровищами

## Особенности реализации

- **Псевдослучайный генератор**: Используется математическая формула на основе синуса вместо модуля `random` для предсказуемых случайных событий
- **Система событий**: Случайные события происходят при перемещении с вероятностью ~10%
- **Контекстная логика**: Команда `solve` автоматически выбирает действие в зависимости от комнаты
- **Награды по комнатам**: Разные комнаты дают разные награды за решение загадок
- **Ловушки с последствиями**: Неверные ответы в комнате с ловушками активируют защитный механизм
